Attention!! Il faut voir très attentivement :
	1- Comment les variables sont déclarées et les classes sont déclarées.
	2- Comment les méthodes de la classe sont écrites.
	3- La forme générale  du programme au niveau esthétique.
	4- Lors d’une déclaration d’un objet, il faut utiliser « new »
	5- L’accès aux attributs d’un objet sont privés, il faut passer par la méthode getters et setters.
	6- Les méthodes sont toujours appelées par un objet.
	7- La fonction « main() » doit être inclue dans une classe.
	
Bon Courage a vous tous.
